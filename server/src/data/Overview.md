# Data Overview

`locations.json`의 209개 장소는 안전·중립·적대 위험도를 구분한다. 초반 초원·은빛그물 숲·늪·화산·갱도부터 Lv.345 월식해구·백야성소까지 이어지며 숲·수렁·화산·황혼왕릉·천둥마루·유리모래 사막·서리잔향 설원·빙경궁·안개파도 해안·침몰왕도·사령묘·철근미궁·역설기계고·잿왕성·공허왕관·월식해구는 분기·순환·합류·퍼즐 경로를 가진다.

서버 시작 시 import 부작용으로 런타임 레지스트리에 들어가는 마스터 데이터를 둔다.

은빛그물 숲 콘텐츠는 사냥활·해독제·늑대 가죽·거미실·독샘을 정의하며 전용 아트 전에는 존재하는 카테고리 fallback을 명시적으로 재사용한다. 황혼왕릉은 언데드 소재 5종, 향약, 네 직업용 중레벨 장비와 5개 조합 제작법을 제공한다. 유리모래 사막은 소재 5종·생존 소모품 2종·직업 장비 5종·6개 조합법을, 서리잔향 설원·빙경궁은 소재 7종·소모품 3종·직업 장비 5종·전승서 2종·7개 조합법을 제공한다. 안개파도 해안·침몰왕도는 소재 8종·소모품 3종·직업 장비 5종·전승서 2종·8개 조합법을 제공한다. 역설기계고는 기계 소재 9종·소모품 4종·직업 장비 5종·전승서 4종·9개 조합법을 기록 중계소 상점과 잇는다. 잿빛성흔 심연은 소재 8종·소모품 3종·직업 장비 5종·전승서 3종·8개 조합법을, 공허왕관 성채와 월식해구는 각각 소재 6종·소모품 2종·직업 장비 5종·전승서 2종·7개 조합법과 두 단계 연속 퀘스트를 각 안전 중계소에 연결한다.

- `items.ts`: 아이템 이미지 key·`baseMetadata`·정의 태그·사용/공격 효과 등록. HP 회복은 `Entity.heal()`로 받는 치유량 상태효과를 반영하고 `restore_survival` 음식·음료는 배고픔/수분을 최대값 안에서 회복하며 `learn_skill` 스킬북은 신규 획득 성공 때만 한 개를 소비한다. 지각 붕괴와 은빛그물 보스의 포식자 도약·은실 사냥망 전승서가 같은 공개 handler를 사용한다. `apply_status_effect`는 metadata의 효과 ID·레벨·시간을 사용하며 전투 강장제·비전 영약·신속의 물약이 60초 버프를 제공한다. 미궁 유물은 전체 스킬 쿨다운 감소·잠기지 않은 인접 길 무작위 전이·다음 공격 확정 회피를 공개 API로 실행한다. Lv.50 이후 직업 무기 4종, 일반 제련 소재 5종과 홍염산지 전용 화맥 광석·홍염강, 조합형 단조 장비 형태 5종, 버프 물약과 미궁 유물은 각 데이터 ID의 전용 128×128 아이콘을 사용한다. 기본 미끼 조합에서 입질까지 약 31~45초가 걸리는 초보자 낚싯대·정교한 균형형 낚싯대, 보물상자 전용 범위/속도 특화 낚싯대, `/낚시` 시 묶음 자동 장착 지렁이 미끼와 `fishingCatalog.ts`의 물고기 아이템도 정의하며 인스턴스는 기본 metadata 전체를 복사하지 않고 top-level delta만 저장한다.
- `projectiles.ts`: 코드에서 참조하는 투사체 마스터 데이터. 물리 화살, 무탄약 지팡이 기본 공격용 마력 구체, 마법사 스킬 전용 마력탄을 각각 독립 정의하고 owner 투사체 가속 반영 계수를 가진다.
- `monsters.ts`: `/몬스터정보`에 쓰는 설명과 Lv.1~345 몬스터의 전투 속성·상태이상·능력치·보상·드롭 정의. 일반 동급 경험치는 `level * 20`이며 성장 관문과 선택 권역 보스는 5배 이상 경험치·고체력·전용 SkillData 또는 미니게임 패턴을 가진다. 월식해구의 리바이어던은 두 기술을 순차 사용하고, 백야대사제는 높은 도발 저항과 치유·보호·제어 위협 가중치로 세 기술을 무작위 사용한다.
- `bossPatterns.ts`: 은빛그물 알주머니, 태양거울·역설 고정자·공허왕관 기둥·백야 조류거울의 생존 여부에 따른 보스 피해 경감과 수정맥·철근·성계 보스의 미니게임 패턴을 소유한다. 조류거울이 하나라도 남아 있으면 백야대사제는 받는 피해를 65% 줄인다.
- `resources.ts`: 비공격 Entity 자원의 능력치, 공격 가능 여부, 도구 제한, 가중치 드롭, 범위 경험치, 상호작용·랜덤 쿨타임 정의. 월식해구의 침은 광맥·조류제단·8~12시간 침수 유물함·백야 조류거울을 포함한다.
- `shops.ts`: 기존 상점과 월식해구 조류관측선 상점의 태그·매매 목록·재고 정의. 지역 상점은 해당 구간 직업 장비·생존품을 판매하고 지역 소재를 희귀도에 따라 매입한다.
- `fishingCatalog.ts`: 일반~신화 각 6종, 총 36종 물고기의 ID·이름·설명·중량·등급 단일 원본.
- `fishing.ts`: 공용 물고기 카탈로그를 낚시 보상 registry에 등록한다.
- `tagEffects.ts`: `source tag → target tag → modifier` 단방향 효과 테이블과 `/속성표`용 라벨·`affinities/*` 아이콘 key. 불·물·얼음·자연·독·전기·돌·어둠·빛·언데드·신성·벌레·금속·땅과 무생물을 모두 매핑한다.
- `statusEffects.ts`: 레거시 상태이상을 현재 source modifier·행동 제한·피해 API로 이식한 마스터 정의. 쇠약의 저주는 공격/마법/치유 효율을, 석화는 전투·이동 행동과 방어 성향을, 열병은 속도·수분 유지를 조절한다. 화염/빙결·열병/빙결·둔화/신속 중화와 저항·해독·보존·발각 단방향 상호작용 표를 함께 소유한다. 전용 아이콘 제작 전에는 의미가 가까운 기존 아이콘을 명시적으로 공유한다.
- `locations.json`: 초반 루미나르부터 월식해구·백야성소까지의 좌표·연결·대표색·통합 오브젝트 원본. 월식해구 24곳은 산호초/침몰선, 해구숲/역조, 성가회랑/조류기관 분기와 hidden 침수보물고 우회로를 가진다. 같은 월드 권역의 장소는 지도에서 하나의 연속 면으로 합쳐지도록 동일한 `mapColor`를 공유한다.
- 은빛그물 숲은 바람결 초원 2에서 분기해 사냥꾼 거점·적갈기 언덕·독실 동굴을 거쳐 안개수렁에 재합류하며, 여왕 산란실의 알주머니 3개가 보스 피해를 35% 경감한다.
- `locations.ts`: Lv.10/20/28/36/45/50/70/90/120/150/180/200/235/275/310 연결 condition handler와 실제 이동 실패 시 공개할 필요 레벨 설명 등록.
- `dungeonPuzzles.ts`: 기존 질문 퍼즐과 월식 조류제단의 해답 Progress, Resource interaction과 hidden 경로 condition을 연결한다.
- `progress.ts`: 게임 이벤트에 연결되는 통계 정의. 공개 치명타 누적과 숨김 검·도끼·활·단검·지팡이 적중 횟수가 있다.
- `progress.ts`의 숨은 `profession:blacksmith` flag는 구형 독립 대장장이 데이터를 정식 직업 슬롯으로 이전하는 호환 값이다.
- `skills.ts`: 아이콘 key·스킬 마스터 metadata·계산 필드·포맷 문자열·Entity 공용 획득/발동/패시브 수명주기와 실제 계수 callback을 공유하는 밸런스 진단 metadata. 공개 기술 계열 표시 metadata와 전사/궁수/암살자/대장장이 0.75초, 모든 마법 0.5초, 동일 화염·빙결·전격 1초의 태그 기반 공유 쿨다운을 정의하며 엘리트 액티브도 메인 계보를 따른다. 다섯 1차 직업은 Lv.30·50 `unlockLevel`에 공격·제어·방어 성장 기술을 자동 획득하고, 지역 보스는 직업 제한 없는 플레이어용 전승 기술을 별도 스킬북으로 제공한다. 역설기계고 전승 기술은 광자창·인과고정·톱니폭우·역설반전이며 시간강 거신과 역설설계자는 별도 몬스터 패턴 SkillData를 사용한다. 전투 본능·매의 눈·살의 감각·마력 순환·체력/감각형 대장장이의 담금질 1차 패시브와 20개 조합별 엘리트 계승 패시브는 직업 조건이 유효할 때 source-key modifier를 적용하고 비활성 시 정리한다. 대장장이는 최대 생명력과 치명타 피해가 함께 반영되는 확정 치명타 `결 파쇄`, 제련 정밀도에 따라 처리량과 캐릭터 경험치가 늘어나는 마력 제련을 사용하며 감각 250 이상이면 본인 제작 장비를 명명하는 `장인의 명명`을 자동 획득한다. 마도 대장장이는 무기 속성에 편향된 화염·맹독·기절·빙결·실명 중 한 적중 효과를 서버 난수로 한 번 부여한다. 은신은 공격·투사체 발사에 즉시 해제되고 바람 회피는 Lv.1 7초부터 레벨당 0.75초 증가한다. 확정 아트 전에는 명시한 카테고리 fallback을 쓰고 시전 메시지가 있는 액티브는 의미가 맞는 기존 4:1 배너를 명시적으로 재사용한다. 효과 설명은 콘셉트·최종값·후속 효과 순서의 문장을 사용하고 피해·보호막 계수는 최종 숫자의 hover에 숨긴다. 발동 조건은 실제 준비할 항목만 표시하며, 주문과 자체 생성 투사체는 장비 독립적으로 발동한다. 직업 지속 버프와 스킬 고유 관통·회피 불가 값은 로테이션 진단에서도 실제 callback과 옵션을 공유한다.
- `jobs.ts`: 전사·궁수·암살자·마법사·감각/체력 중심 대장장이 5개 1차 직업, 각 직업의 상시 패시브를 포함한 지급 스킬·메인/서브 modifier와 동일 직업을 제외한 순서 있는 20개 Lv.200 엘리트 조합 및 조합 전용 계승 패시브·액티브 지급. 엘리트의 공통 주능력치 증폭은 전용 패시브·액티브와 중복 폭주하지 않도록 15%를 사용한다.
- `crafting.ts`: predicate 재료·발견 조건·제작 시간·결과 factory 정의. 지역 조합법에 월식해구 7개를 더한다.
- `npcs.ts`: NPC ID, 조건부 진입점과 generator 대화 장면 정의. 조류항해사 미레나는 관측선 복구와 백야 종결 연속 퀘스트를 안내한다.
- `quests.ts`: NPC 제공/보고 ID, 단계형 목표와 lazy 표시명 보상 정의. 튜토리얼·전직·기존 지역 연속 퀘스트에 월식해구 관측선/백야 종결 연속 퀘스트를 더한다.

ID는 DB와 다른 데이터 파일에서 참조되므로 변경 시 참조 전체를 확인한다. 데이터 구조나 ID가 바뀌면 이 문서와 해당 [`docs/systems/`](../../../docs/systems) 문서를 갱신한다.
