# 직업·전직 시스템

## 구조와 해금

`JobData`는 코드 마스터 정의이며 `defineJob/getJob/getAllJobs` 정적 레지스트리로 찾는다. `JobTier`는 1차/엘리트, `JobSlotType`은 메인/서브와 요구 레벨을 소유하는 클래스형 enum이다. Player의 `CareerProfile`은 내부 Progress key를 숨기고 직업 선택·계보 호환·능력치 modifier·스킬 지급·엘리트 전직을 담당한다.

- 메인 직업: Lv.20부터 전직소 퀘스트 완료로 선택한다.
- 서브 직업: Lv.50부터 전직소 퀘스트 완료로 선택한다.
- 메인과 같은 직업은 서브로 선택할 수 없다. `CareerProfile.canAssign/assign`과 퀘스트 조건 양쪽에서 거부한다.
- 엘리트 직업: Lv.200에 서로 다른 메인→서브의 순서 있는 조합으로 자동 전직한다.
- 4개 1차 직업의 서로 다른 조합 12개가 모두 결과를 가지며 `전사→마법사`와 `마법사→전사`는 다른 엘리트 직업이다.
- 각 엘리트 직업은 전직 즉시 조합 전용 계승 패시브 하나를 추가로 획득한다. 기존 메인·서브 패시브도 함께 유지된다.

## 영속성과 계보

메인·서브·엘리트 ID는 `career:main_job`, `career:sub_job`, `career:elite_job` Progress STATE로 저장한다. 변경은 메모리 dirty 상태로 남고 기존 30초/unload/종료 flush를 사용한다. 퀘스트 보고와 엘리트 전직은 중요한 변경이므로 `Player.save()`도 요청한다. 별도 Prisma migration은 없다.

엘리트 전직 뒤에도 원래 메인 ID와 서브 ID를 보존한다. `CareerProfile.hasJob(jobId, slot?)`은 엘리트의 `parentJobIds`를 따라 원래 메인 계보를 호환하므로, 마검사는 전사 메인 스킬과 마법사 서브 스킬을 계속 표시·사용할 수 있다. 스킬은 박탈하지 않고 `SkillData.jobRequirement`와 `isVisible/checkUsable`에서 표시·사용만 제어한다.

## 초기 직업과 스킬

- 전사: 패시브 `전투 본능`(공격력 +6%, 방어력 +6), 강철 베기, 전투 질주, 불굴. 검/도끼와 생명력·공격·속도 균형.
- 궁수: 패시브 `매의 눈`(치명타 확률 +4%p, 이동속도 +5%), 마력 화살, 다중 사격, 충격 화살, 바람 회피. 활 투사체·기절·임시 확정 회피.
- 암살자: 패시브 `살의 감각`(치명타 피해 +18%p, 방어 관통력 +4), 은신, 암습, 맹독 칼날. 단검·은신·회피 불가 치명타·맹독.
- 마법사: 패시브 `마력 순환`(마법력 +7%, 정신력 재생 +1.5/초), 마력탄, 마력 보호막, 원소 속박, 원소 통찰. 지팡이·정신력·마법 방어와 제어.

패시브도 일반 스킬과 같은 `SkillData`/`SkillBook` 경계를 사용한다. 보유한 패시브는 스킬 목록과 정보창에 표시하지만 직접 사용 버튼과 전투 퀵 HUD에서는 제외한다. 직업 또는 계보 조건이 유효할 때만 source-key 능력치 modifier를 유지하며, 직업을 잃거나 스킬이 회수되면 `onPassiveInactive`가 즉시 정리한다. 전용 패시브 아이콘은 후속 아트 작업 전까지 각 직업 아이콘을 fallback으로 사용한다.

엘리트 계승 패시브는 추격의 검로·그림자 칼날·마력 검로·공성 자세·야간 포착·원소 조준·처형 준비·환영 사격술·비전 수확·전투 마력갑·성좌 유도·주술 각인이다. 각 효과는 주 직업의 핵심 공격 능력과 서브 직업의 속도·치명타·생존·마법 능력 중 하나를 결합하며, `/직업밸런스`의 Lv.200 시나리오도 실제 callback을 적용한다.

궁수의 물리·마법 화살과 마법사의 공격 마법 투사체는 발사 순간 시전자의 치명타 확률·치명타 피해를 복사해 적중 시 치명타를 판정한다.

직업 스킬은 `/스킬 이름`과 `activationMessage`/`activationPhrase`의 `이름!` 일반 메시지로 발동한다. `SkillData.weaponRequirement`는 `Equipment.hasEquippedItemTag`로 주무기 태그를 검사한다. 전직 보상으로 전사는 훈련용 도끼, 궁수는 가벼운 활, 암살자는 독 단검, 마법사는 견습 지팡이를 받는다.

전투 질주·불굴·바람 회피·은신·마력 보호막·원소 통찰은 발동 시 시전자 본인에게만 시전 문구를 남기고, 현재 레벨 기준 상승량과 지속시간도 본인 전용 상세 메시지 및 notification으로 알린다.

직업 스킬의 공격·버프 수치는 스킬 레벨에 따라 성장한다. 설명 본문에는 현재 플레이어 능력치와 스킬 레벨로 계산한 결과만 간결하게 표시하고, 수치 hover에는 `공격력 × 계수`, 기본값, 레벨당 증가량을 표시한다. 설명과 실제 발동은 `skills.ts`의 `valueByLevel/percentByLevel` 계산을 함께 사용한다. 직업 소유·엘리트 하위 호환 판정은 내부 사용 조건으로 유지하되 발동 조건 문구에는 반복 노출하지 않는다.

마법사는 불·얼음·전기 속성 몬스터 처치 통계를 각각 5회 달성하면 화염구·빙결탄·뇌전구를 `autoAcquire`로 획득한다. 해당 스킬도 마법사 또는 마법사 계보 엘리트/서브 호환 조건을 유지한다.

## UI와 콘텐츠

루미나르 광장에서 `루미나르 전직소`로 이동해 전직관 세레나와 대화한다. 전직 시험은 슬롯·직업별 8개 퀘스트이며 한 시험을 수락하면 완료/포기 전까지 NPC가 진행 상태를 안내한다. `/직업(job, career)`은 현재 직업과 다음 해금, `/직업정보(ji)`는 직업 설명과 지급 스킬을 보여준다. `/상태창` 기본 정보에도 실제 메인/원래 계보와 서브 직업이 표시된다.
