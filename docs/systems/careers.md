# 직업·전직 시스템

## 구조와 해금

`JobData`는 코드 마스터 정의이며 `defineJob/getJob/getAllJobs` 정적 레지스트리로 찾는다. `JobTier`는 1차/엘리트, `JobSlotType`은 메인/서브와 요구 레벨을 소유하는 클래스형 enum이다. Player의 `CareerProfile`은 내부 Progress key를 숨기고 직업 선택·계보 호환·능력치 modifier·스킬 지급·엘리트 전직을 담당한다.

- 메인 직업: Lv.20부터 전직소 퀘스트 완료로 선택한다.
- 서브 직업: Lv.50부터 전직소 퀘스트 완료로 선택한다.
- 메인과 같은 직업은 서브로 선택할 수 없다. `CareerProfile.canAssign/assign`과 퀘스트 조건 양쪽에서 거부한다.
- 엘리트 직업: Lv.200에 서로 다른 메인→서브의 순서 있는 조합으로 자동 전직한다.
- 전사·궁수·암살자·마법사·대장장이 5개 1차 직업의 서로 다른 조합 20개가 모두 결과를 가지며 `전사→대장장이`와 `대장장이→전사`도 서로 다른 엘리트 직업이다.
- 각 엘리트 직업은 전직 즉시 조합 전용 계승 패시브와 액티브 스킬을 하나씩 추가로 획득한다. 기존 메인·서브 스킬과 패시브도 함께 유지된다.

## 영속성과 계보

메인·서브·엘리트 ID는 `career:main_job`, `career:sub_job`, `career:elite_job` Progress STATE로 저장한다. 변경은 메모리 dirty 상태로 남고 기존 30초/unload/종료 flush를 사용한다. 퀘스트 보고와 엘리트 전직은 중요한 변경이므로 `Player.save()`도 요청한다. 별도 Prisma migration은 없다.

엘리트 전직 뒤에도 원래 메인 ID와 서브 ID를 보존한다. `CareerProfile.hasJob(jobId, slot?)`은 엘리트의 `parentJobIds`를 따라 원래 메인 계보를 호환하므로, 마검사는 전사 메인 스킬과 마법사 서브 스킬을 계속 표시·사용할 수 있다. 스킬은 박탈하지 않고 `SkillData.jobRequirement`와 `isVisible/checkUsable`에서 표시·사용만 제어한다.

## 초기 직업과 스킬

- 전사: 패시브 `전투 본능`(공격력 +6%, 방어력 +6), 강철 베기, 전투 질주, 불굴. 검/도끼와 생명력·공격·속도 균형.
- 궁수: 패시브 `매의 눈`(치명타 확률 +4%p, 이동속도 +5%), 마력 화살, 다중 사격, 충격 화살, 바람 회피. 활 투사체·기절·임시 확정 회피.
- 암살자: 패시브 `살의 감각`(치명타 피해 +18%p, 방어 관통력 +4), 은신, 암습, 맹독 칼날. 단검·은신·회피 불가 치명타·맹독.
- 마법사: 패시브 `마력 순환`(마법력 +7%, 정신력 재생 +1.5/초), 마력탄, 마력 보호막, 원소 속박, 원소 통찰. 정신력·마법 방어와 제어. 지팡이는 마법력 기반 투사체 평타와 마법력·투사체 가속 증폭을 제공하지만 주문 사용을 제한하지 않는다.
- 대장장이: 패시브 `대장장이의 담금질`(최대 중량 +10, 최대 생명력 +8%, 치명타 피해 +20%p), 확정 치명타 `결 파쇄`, 마력 제련, 금속 단조. 체력으로 버티고 감각으로 금속의 결·급소·단조 박자를 읽는 생산·근접 혼합 직업이다.

패시브도 일반 스킬과 같은 `SkillData`/`SkillBook` 경계를 사용한다. 보유한 패시브는 스킬 목록과 정보창에 표시하지만 직접 사용 버튼과 전투 퀵 HUD에서는 제외한다. 직업 또는 계보 조건이 유효할 때만 source-key 능력치 modifier를 유지하며, 직업을 잃거나 스킬이 회수되면 `onPassiveInactive`가 즉시 정리한다. 전용 패시브 아이콘은 후속 아트 작업 전까지 각 직업 아이콘을 fallback으로 사용한다.

엘리트 계승 패시브는 기존 12종과 대장장이 조합 8종을 합쳐 20종이다. 대장장이 조합은 장인의 무기술·정밀 기공·연마된 살의·전투 룬각·전장의 담금질·자동 조준 장치·독금 연마·마도 제련으로 공격·방어·투사체·독·마법 능력을 생산 기술과 결합한다. `/직업밸런스`의 Lv.200 시나리오도 모든 조합의 실제 callback을 적용한다.

같은 엘리트 직업이 지급하는 계승 패시브와 액티브는 스킬 목록에서 구분되는 고유 표시 이름을 사용한다. 비전 사신은 패시브 `영혼 포식`으로 마법력·치명타 피해를 높이고, 액티브 `비전 수확`으로 대상을 공격한다.

엘리트 액티브는 기존 12종에 완성의 일격·철우 연사·톱날 급습·폭발 룬·모루 강타·자동 추적탄·독금 천공·마도 용융탄을 더한 20종이다. 조합에 따라 회피 불가, 확정 치명타, 속성 투사체, 마비독·맹독, 보호막을 기존 전투 API로 조합한다. 전용 아이콘 제작 전까지 주계열 직업 또는 채굴 도구 카테고리 아이콘을 fallback으로 사용한다.

대장장이를 메인, 마법사를 서브로 선택한 `마도 대장장이`는 공통 계승 패시브·액티브 외에 후가공 유틸리티 `마법 부여`도 획득한다. 이 스킬은 `/마법부여` 명령에서 사용하며 무기 속성에 편향된 적중 상태효과를 영속 인스턴스 delta로 한 번 확정한다. 아직 전용 후가공 스킬 아트가 없으므로 마법력 아이콘을 fallback으로 사용하며 TODO로 관리한다.

궁수의 물리·마법 화살과 마법사의 공격 마법 투사체는 발사 순간 시전자의 치명타 확률·치명타 피해를 복사해 적중 시 치명타를 판정한다.

직업 스킬은 `/스킬 이름`과 `activationMessage`/`activationPhrase`의 `이름!` 일반 메시지로 발동한다. `SkillData.weaponRequirement`는 검·도끼로 직접 타격하거나 활로 사격하는 등 무기가 기술 동작에 반드시 필요할 때만 `Equipment.hasEquippedItemTag`로 주무기 태그를 검사한다. 마법 주문, 환영 단검·자동 추적탄처럼 스킬이 직접 만드는 투사체에는 무기 제한이 없다. 따라서 서로 다른 메인·서브 직업의 무기를 장착해도 다른 계열의 비무기 스킬을 함께 사용할 수 있다. 전직 보상으로 전사는 훈련용 도끼, 궁수는 가벼운 활, 암살자는 독 단검, 마법사는 견습 지팡이를 받는다.

전투 질주·불굴·바람 회피·은신·마력 보호막·원소 통찰은 발동 시 시전 문구와 현재 레벨 기준 상승량·지속시간을 상세 메시지 및 notification으로 알린다. 솔로에서는 본인에게만 보이고 파티 중에는 파티 전투 피드로 공유된다.

직업 스킬의 공격·버프 수치는 스킬 레벨에 따라 성장한다. 설명 본문에는 현재 플레이어 능력치와 스킬 레벨로 계산한 결과만 간결하게 표시하고, 수치 hover에는 `공격력 × 계수`, 기본값, 레벨당 증가량을 표시한다. 설명과 실제 발동은 `skills.ts`의 `valueByLevel/percentByLevel` 계산을 함께 사용한다. 직업 소유·엘리트 하위 호환 판정은 내부 사용 조건으로 유지하되 발동 조건 문구에는 반복 노출하지 않는다.

마법사는 불·얼음·전기 속성 몬스터 처치 통계를 각각 5회 달성하면 화염구·빙결탄·뇌전구를 `autoAcquire`로 획득한다. 해당 스킬도 마법사 또는 마법사 계보 엘리트/서브 호환 조건을 유지한다.

## UI와 콘텐츠

루미나르 광장에서 `루미나르 전직소`로 이동해 전직관 세레나와 대화한다. 전직 시험은 슬롯·직업별 10개 퀘스트이며 한 시험을 수락하면 완료/포기 전까지 NPC가 진행 상태를 안내한다. `/직업(job, career)`은 현재 직업과 다음 해금, `/직업정보(ji)`는 직업 설명과 지급 스킬을 보여준다. `/상태창` 기본 정보에도 실제 메인/원래 계보와 서브 직업이 표시된다.

대장장이는 `career:blacksmith` 정식 1차 직업이며 전직관 세레나에게 다른 직업과 동일하게 선택할 수 있다. 메인 시험은 광맥 5개, 서브 시험은 광맥 10개를 파괴하며 메인 전직 보상으로 철 곡괭이를 받는다. 은맥 광부 보급소의 로안이 제공하는 `불꽃 없는 제련법`도 기존 별도 진입 경로로 유지된다. 메인이 비어 있으면 Lv.20부터 메인으로, 다른 메인이 이미 있으면 Lv.50부터 서브로 전직하며 두 슬롯이 모두 차 있으면 수락할 수 없다. 구형 `profession:blacksmith` 플래그는 로그인 시 기존 직업을 덮어쓰지 않고 빈 슬롯으로 이전하기 위한 숨은 호환 데이터로만 남는다. 별도 Prisma migration은 필요 없다.

대장장이 권장 배분은 감각:체력:근력:정신력 `4:4:1:1`이다. 감각은 일반 치명타 능력과 함께 `제련 정밀도`를 1당 0.15% 올리고, 결 파쇄는 공격력+최대 생명력 합산값에 치명타 피해를 곱해 감각과 체력을 동시에 전투력으로 바꾼다. 메인 대장장이는 최대 생명력·방어·치명타 확률을, 서브 대장장이는 최대 생명력·방어를 보강한다.
