# 콘텐츠 규모 추적

레거시 저장소의 콘텐츠 밀도를 현재 서버 권위형 구조로 옮길 때 사용하는 수량 기준과 진행 상태다. 레거시의 이름·코드를 그대로 복사하지 않고, 현재 태그·상태효과·위협 AI·퍼즐·상점·퀘스트 API로 변형한다.

## 2026-07-23 기준

| --- | ---: | ---: | --- |
| 장소 | 233 | 233 | 목표 규모 도달. 역근수해 이후 월드 확장은 보류한다. |
| 몬스터 | 65 | 137 | 양적 기준 도달. 이후에는 기존 전투의 완성도를 우선한다. |

현재 추가가 완료된 선택 권역은 은빛그물 숲부터 역근수해·태초심장까지다. 마지막 권역은 Lv.345~380의 24개 분기 장소, 순차 낙하·포자 패턴의 역근 포식수, 무작위 박동·망각 패턴의 태초심장, 첫 기억의 제단 뒤 숨은 유물고와 파괴 전 태초심장 피해를 70% 줄이는 심장씨앗을 연결한다. 장소 수는 레거시 목표 233곳에 도달했다. 직업 성장기도 다섯 1차 직업의 Lv.75/100/140/180 기술과 화염·빙결·전격 3단계 상위 숙련 분기를 채워 1차 확장 목표를 완료했다.

## 확장 우선순위

1. 다음 콘텐츠 확장 전에는 `BALANCE_PROFILE_LEVELS`의 Lv.20/50/75/100/140/180/200 회귀 결과를 먼저 확인한다.
2. 관리자 밸런스 명령·페이지와 `balance:report`의 동레벨 일반/보스 프로필에서 회피율·관통 후 방어·처치시간·평타 비중을 확인한다.
3. 1차 직업 Lv.75~180과 Lv.200 엘리트 조합의 보스 DPS 최고/최저 편차는 자동 검증 상한인 1.5배를 넘기지 않는다.
4. 확정 전용 아트 대신 존재하는 카테고리 fallback을 사용하고 교체 TODO를 유지한다.

## 검증 기준

- `server/src/data/world.test.ts`가 장소 양방향 연결, 오브젝트 ID, 권역 대표색, 상점·제작법·퀘스트·퍼즐·보물·보스 패턴을 권역 단위로 검증한다.
- `server/src/scripts/validateMasterData.ts`가 모든 참조와 명시된 fallback 아이콘 경로를 검증한다.
- `server/src/scripts/reportBalance.ts`가 공용 레벨 구간과 황혼왕릉부터 역근수해까지의 장비를 같은 직업/레벨 기준선에서 비교하고, 지역 전승 기술의 회피·관통·자원·쿨다운을 함께 출력한다.
- `server/src/models/Balance.test.ts`가 상위 1차 직업과 순서 있는 엘리트 20조합의 보스 DPS 편차 1.5배, 평타 피해 비중 15% 이상을 회귀 검증한다.
- 수량은 문서의 추정이 아니라 현재 마스터 데이터를 실제로 import한 런타임 레지스트리 수와 `locations.json` 길이로 갱신한다.
